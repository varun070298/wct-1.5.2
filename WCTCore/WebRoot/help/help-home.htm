Version v1.1 GA

Official General Availability release
Known issues
  • Exception thrown by Review Tool when Anchor Tag Attribute is Empty.
    The Wayback TagMagix class throws java.lang.StringIndexOutOfBoundsException if a tag attribute is empty. This prevents the site being successfully reviewed. The Web Curator Team are investigating whether a newer release of the Wayback tool resolves this issue. As an example, if a reviewed HTML page contains the code:
    <a href=""></a>
    This will throw an exception during the review phase since the HREF attribute is empty. This issue only impacts tags that are rewritten by the TagMagix class from the Wayback project. SourceForge Reference #1541022.
  • Harvesting of websites that use SSL encryption results in Heritrix storing the encrypted data in the ARC file. This means that any Harvested https site will not be reviewable.
