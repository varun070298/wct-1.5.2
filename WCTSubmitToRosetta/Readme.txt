This module WCTSubmitToRosetta archives a WCT harvest into ExLibris Rosetta application. This module was developed and is currently being used by National Library of New Zealand to preserve web harvests.

Please see the build notes in the "build" directory until we update the main WCT documentation to include further details about this module.
